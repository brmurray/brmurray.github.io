Sample Work

Grid Topology Reconstruction

The relationship of an electric power distribution network's electrical connectivity and the geographic position is not always available, and the reconstruction is not obvious. I wrote a bit of code to reconstruct an electrical connectivity map from GIS data of a medium-voltage feeder from SouthernSouthern California Edison's territory in Goleta, California. Explore the resulting map on Github.

Renewable Hosting Capacity Analysis

The grid can add lots more distributed energy resources (DERs) generally, but each node in the grid is ultimately constrained by a unique set of operation conditions. SCE provides some data about these constraints, and I made a tool to explore the profile of the nodes within my feeder of interest. See the details on Github.
